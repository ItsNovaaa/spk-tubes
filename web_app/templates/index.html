Analisis Keputusan Multikriteria dengan FUCOM & VIKOR

Sistem ini mengkombinasikan metode pembobotan kriteria **FUCOM (Full Consistency Method)** dan metode perankingan alternatif **VIKOR (ViseKriterijumska Optimizacija I Kompromisno Resenje)** untuk mencari solusi kompromi terbaik dalam berbagai permasalahan keputusan yang rumit.

Panduan Penggunaan Aplikasi

1

Format File Excel

Pastikan file Excel memiliki baris pertama sebagai header (nama alternatif dan kriteria). Alternatif biasanya berada di kolom pertama. Seluruh kolom penilaian kriteria harus bernilai angka.

2

Tentukan Sifat Kriteria

Kriteria **Benefit** menandakan semakin tinggi nilai semakin baik (misalnya IPK, Prestasi). Kriteria **Cost** menandakan semakin rendah nilai semakin baik (misalnya Tagihan Listrik, Income).

3

Pilih Mode Bobot FUCOM

- **Objective**: Bobot dihitung otomatis dari rata-rata data.
- **Manual**: Masukkan bobot manual Anda.
- **Subjective**: Urutkan prioritas kriteria dan masukkan rasio $\varphi$.
